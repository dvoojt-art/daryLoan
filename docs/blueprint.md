# **App Name**: LendMate

## Core Features:

- Member Authentication & Role Gating: Secure login system with dual-access roles: Admin for full system control and Member for personal record viewing and requests.
- Member Loan Portal: A streamlined interface for members to submit loan requests, view repayment schedules, and monitor approval status.
- AI Loan Risk Tool: Generative AI tool that analyzes member contribution history and requested amounts to provide admins with a risk assessment summary for faster decision making.
- Admin Approval Workflow: A management pipeline for administrators to review, approve, or reject loan applications with one click.
- Contribution & Collection Ledger: A persistent database records system to track member contributions and loan collection progress.
- Smart Reporting Engine: Dashboard with automated financial reporting using logic to calculate total disbursements, collections, and outstanding balances.
- Excel-Sync Formula Inputs: Dynamic smart forms for admins that use pre-defined formulas to automate payment calculations without manual entry.

## Style Guidelines:

- Primary color: Sophisticated Indigo (#3148B1) evoking trust and professionalism.
- Background color: Ultra-clean Light Sky (#F6F7FA), heavily desaturated for a modern, breathable fintech appearance.
- Accent color: Professional Teal (#2889A1) used for buttons and status indicators to create a sharp contrast.
- Font pairing: 'Space Grotesk' for headlines to give a modern, machined feel, and 'Inter' for readability in high-density data tables and body text.
- Code font: 'Source Code Pro' for displaying formulaic fields and interest rate calculations.
- Minimalist, thin-line icons used sparingly to highlight primary navigation categories and transaction statuses.
- Crisp transitions between the member list and detailed reports to maintain a smooth, non-disruptive user experience.
